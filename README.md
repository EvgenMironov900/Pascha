# 📆 Православный календарь переходящих праздников v1.0

Простое веб-приложение для расчёта дат православных праздников, зависящих от Пасхи (Масленица, Великий пост, Вербное воскресенье, Вознесение, Троица и др.).

## 🚀 Использование

1. Откройте [страницу приложения](https://evgenmironov900.github.io/Pascha/) или  
      https://evgenmironov900.github.io/Pascha/
2. Введите интересующий вас год
3. Нажмите «Показать» или клавишу Enterз

## 🥚 Доступные праздники

| Праздник / событие | Относительно Пасхи |
|-------------------|-------------------|
| Масленица | за 55–49 дней |
| Прощёное воскресенье | за 49 дней |
| Великий пост | за 48–1 день |
| Вербное воскресенье | за 7 дней |
| Пасха | 0 |
| Красная горка (Антипасха) | +7 дней |
| Радоница (родительский день) | +9 дней |
| Жёны-мироносицы | +14 дней |
| Вознесение Господне | +40 дней |
| Троица (Пятидесятница) | +49 дней |
| День Святого Духа | +50 дней |
| Петров пост | +57 дней до 11 июля |

## 🎯 Особенности

- Работает полностью в браузере, без сервера
- Мгновенный расчёт при вводе года
- Адаптивный дизайн для смартфонов и планшетов
- Дата Пасхи вычисляется по формулам К. Гаусса (православная версия)
- Все даты выводятся в григорианском календаре

### 📱 Мобильное совершенство
    - Добавляйте на рабочий экран ссылку из браузера - иконка "Пасхалия" отображается идеально!
    - Полностью адаптивный сенсорный интерфейс

## 🌐 Автономная работа

Pascha создан как **полностью автономный инструмент**, работающий где угодно после скачивания.

### 📂 Как использовать оффлайн:
    1. Скачайте `index.html` из репозитория
    2. (Опционально) Переименуйте в `Pascha.html` для удобства
    3. Откройте в любом современном браузере
    4. Всё! Интернет не нужен

#### 🏢 Идеально для:
        - **Корпоративных/государственных сетей** без доступа в интернет
        - **Образовательных учреждений** с ограниченным подключением
        - **Полевых исследований** и удалённых локаций
        - **Защищённых сред**, где облачные сервисы запрещены
        - **Устаревших систем** с только базовым браузером
        - **Режима полёта** и путешествий без Wi-Fi
